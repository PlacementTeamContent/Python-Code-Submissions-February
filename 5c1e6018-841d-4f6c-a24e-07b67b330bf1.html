ASCII Sum


Calculate the sum of ASCII values for each character in a given string txt and print the sum.



- Input string txt contains only uppercase English letters.
- Calculated value is within the integer range.



Input Format


    

  • Input is a single line, which contains a string txt.
    • 





Output Format


    

  • Print the calculated integer value.
    • 





Constraints


    

  • 0 <=length(txt)<= 50
    • 






Explanation


Input


AB


Output


131


The input string contains two letters A and B.
A ASCII equivalent is 65
B ASCII equivalent is 66 
Therefore, Sum = 65 + 66 = 131
#Write your code here
n=input()
length=len(n)
add=0
if 0<=length<=50:
    for i in range(length):
        add+=ord(n[i])
print(add)

    • 

  • The second line of each test case consists of the second string.
    • 



Output Format


    

  • T lines containing YES or NO for each test case.
    • 





Constraints


    

  • Each word will consist of English alphabets which will be a mix of upper and lower case alphabets.
    • 





Example


Input


2
LISTEN
silent
PRESENT
PARENTS




Output


YES
NO




Explanation


Testcase 1
word1 = LISTEN
word2 = silent

Testcase 2

#Write your code here
n=int(input())
new=[]
res=[]
for i in range(2*n):
    new+=[input().lower()]
num=0
for i in range((2*n)-1):
    for j in new[0]:
        for k in new[i+1]:
            if j==k:
                num+=1
    if i==len(new[i]):
        break
    elif num==len(new[i]):
        res+=["YES"]
    else:
        res+=["NO"]

    print(res[0])
else:

1. The character occurrence should be sorted based on their first appearance in the text and printed in lowercase.
2. Input text is case-insensitive.




Input Format


The input contains a string representing text.




Output Format


The output should be in the format of "character-occurrence count" separated by new lines.




Example


Let's consider the sample input:


ProgrammingFun


Here the given input is "ProgrammingFun". The characters that occur more than once are r, g, m and n.
So, the output should be:


r-2
g-2
m-2
n-2


#Write your code here
n=input().lower()
group=sorted(set(n))
for i in group:
    count=n.count(i)
    if count>1:
        print(i+"-"+str(count))
