Vowel Count


Given a string txt, count the total number of vowels (A, E, I, O, U) in the string and print the count.



- Input string txt contains only uppercase English letters.
- Calculated value is within the integer range.



Input Format


    

  • Input is a single line, which contains a string txt.
    • 





Output Format


    

  • Print the total count of vowels.
    • 





Constraints


    

  • 0 <=length(txt)<= 50
    • 






Explanation


Input


HELLO


Output


2


The input string contains one E and one O
Total number of vowels = 2
# Write your code here
s=input()
vowels="AEIOU"
c=0
for i in s:
    if i in vowels:
        c+=1

- If multiple characters have the same frequency, choose the character with the lower ASCII value.
- The replacement character will be between 'a' and 'z'.
- All characters in text are lowercase.




Input Format


The input contains two lines:


    

  • The first line contains the string text.
    • 

  • The second line contains the character replacement.
    • 





Output Format


The output is a single string after replacement.




Constraints


    

  • 0 <= length of string <= 10^5
    • 





Example


Input:


aaabbaaacc
p


Output:


pppbbpppcc


Explanation:
The character 'a' appears most frequently in text, and replacing 'a' with 'p' gives pppbbpppcc.
#Write your code here
s=input()
a=input()
d={}
for i in s:
    if i not in d:
        d[i]=1
    else:
        d[i]+=1 
req=""
for values in d:
    req+=str(max(values))
    break 
res=s.replace(req,a)
print(res)
