Check Rearrangement


Write a program to check RearrangeStrings that takes two strings as input from the user and determines if one string can be rearranged to match the other. RearrangeStrings are strings that have the same characters with the same frequency but in any order. You should print YES if the input strings are RearrangeStrings, otherwise print NO.



1. The program should treat strings as case-insensitive when determining if they are RearrangeStrings. For example, DESSERT and STRESED should be recognized as RearrangeStrings.





Input Format


    

  • First will have the number of test cases T.
    • 

  • The first line of each test case consists of the first string.
    • 

  • The second line of each test case consists of the second string.
    • 



Output Format


    

  • T lines containing YES or NO for each test case.
    • 





Constraints


    

  • Each word will consist of English alphabets which will be a mix of upper and lower case alphabets.
    • 





Example


Input


2
DESSERT
STRESED
CIPHER
RHEPICS




Output


YES
NO




Explanation


Testcase 1
word1 = DESSERT
word2 = STRESED
Both words contain the same alphabets, so they can be rearranged.


Testcase 2
word1 = CIPHER
word2 = RHEPICS
Here, CIPHER and RHEPICS have different characters so they cannot be rearranged.


#Write your code here
num_user_str=int(input())
for i in range(num_user_str):
    b=list(input().lower())
    c=input().lower()
    if len(b)==len(c):
        for i in c:
            if i in b: 
                b.remove(i)
        if len(b)==0:
            print("YES")
        else:
            print("NO")
    else:
        print("NO")

2. All inputs are given in lowercase letters only.

def converting_numerically(number):
    if number=="one":
        return "1" 
    elif number=="four":
        return "4" 
    elif number=="two":
        return "2" 
    elif number=="three":
        return "3" 
    elif number=="five":
        return "5" 
    elif number=="six":
        return "6" 
    elif number=="seven":
        return "7" 
    elif number=="eight":
        return "8" 
    elif number=="nine":
        return "9"

        return "0"

- If there is a tie in frequency, the character with a lower ASCII value should be chosen.
- Character new_char is between 'a' and 'z'.
- All characters in input_string are lowercase letters.




Input Format


The input consists of two lines:


    

  • The first line contains the string input_string.
    • 

  • The second line contains the character new_char.
    • 





Output Format


The output is the modified string after replacement.




Constraints


    

  • 0 <= length of string <= 10^5
    • 





Example


Input:


cdefcccfc
m


Output:


mdefmmmfm


Explanation:
The character 'c' appears most frequently in input_string, which when replaced with 'm' results in mdefmmmfm.
#Write your code here
user_string=input().lower()
user_char=input()
char_list=[]
count_list=[]
rep_char_count=[]
for i in user_string:
    if i not in char_list:
        char_list.append(i)
        count_list.append(user_string.count(i))
b=max(count_list) 
if count_list.count(b)==1 :
    replace_char=char_list[count_list.index(b)]
    print(user_string.replace(replace_char,user_char))
else:
    for i in range(count_list.count(b)):
        rep_char_count.append(char_list[count_list.index(b)])
        char_list.remove(char_list[count_list.index(b)])
    rep_char_count.sort()
    replace_char=rep_char_count[0]
    print(user_string.replace(replace_char,user_char))
