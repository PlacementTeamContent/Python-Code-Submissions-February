Phone Digits


You're developing a software for converting phone numbers given verbally into their numeric equivalent for efficient data recording. The department often receives phone numbers spelled out digit by digit.

1. Contact numbers should always be ten digits long. Return "invalid input" if this is not the case.
2. The entire input is in lowercase.

Input Format

The input is a line containing the phone number in words separated by spaces. 

Output Format

Output the numeric format of the phone number in a single line. 

Example


nine five six three three zero five nine two one


Output


9563305921


Explanation

The sequence of words "nine five six three three zero five nine two one" is converted into the numeric form 9563305921.


a = input().split()
number = ""
if len(a) == 10:
    for i in a:
        if i == "one":
            number += "1"
        elif i == "two":
            number += "2"
        elif i == "three":
            number += "3"
        elif i == "four":
            number += "4"
        elif i == "five":
            number += "5"
        elif i == "six":
            number += "6"
        elif i == "seven":
            number += "7"
        elif i == "eight":
            number += "8"
        elif i == "nine":
            number += "9"
        else:
            number += "0"
    print(number)
else:
    print("invalid input")

Output Format

        name += [i]

Maximum Unique Length


Given a string S consisting of lowercase alphabets, write a program to find the length of the longest substring with unique characters in S.


Input Format


The input will be a single string S.


Output Format


The output will be an integer representing the length of the longest substring with unique characters.


Constraints


    

  • 0 <= S.length <= 10^8
    • 



Explanation


Let's take an example:


devawq


In this case, the input "devawq" has the longest substring with unique characters as "devawq". The length of "devawq" is 6. Hence, the output is 6.
# print(len(set(input())))
a = input()
subsets = []

for i in range(len(a)):
    already_used = ""
    subset = ""
    for j in range(i , len(a)):
        if a[j] not in already_used:
            subset += a[j]
            already_used += a[j]
        else:
            break
    subsets.append(subset)
subsetss = [len(set(i)) for i in subsets]
print(max(subsetss))
